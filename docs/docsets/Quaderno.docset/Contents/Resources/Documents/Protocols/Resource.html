Resource

public protocol Resource

Requirements of a resource.

  • name

    Name of a resource.

    This value is used to build URIs that expose different operations over the same resource. For instance, all operations pertaining to invoicing may use the name "invoices", thus having these URIs:

    • https://quadernoapp.io/api/v1/invoices
    • https://quadernoapp.io/api/v1/invoices/1.json
    • https://quadernoapp.io/api/v1/invoices/deliver/24.json
